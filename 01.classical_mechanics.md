## Pendulum Waves

Let's start by modeling a single pendulum:

:::{figure}
:::

The equation of motion for a pendulum is given by (we'll derive this later):

:::{math}
\theta''(t) = -\frac{g}{L}\sin(\theta(t))
:::

### Small-angle Approximation

To solve this analytically, you probably made the "small amplitude" approximation in 8.01. For small amplitudes of θ, we may replace sin(θ(t))->θ(t).

:::{wolfram}
:echo: true

DSolveValue[\[Theta]''[t] + g/L \[Theta][t] == 0, \[Theta][t], t]
:::

This is a 2^nd order ODE. We therefore need 2 initial conditions to specify the two constants above:

- We'll set the initial displacement at a symbolic value $\theta(0) \to \theta_0$
- and the initial velocity to zero  $\theta'(0) \to 0$

:::{wolfram}
:echo: true

smallDisplacementSymbolicSolution[\[Theta]0_, g_, L_][t_] = 
 DSolveValue[{\[Theta]''[t] + g/L \[Theta][t] == 
    0, \[Theta][0] == \[Theta]0, \[Theta]'[0] == 0}, \[Theta][t], t]
:::

:::{note} Coding comment
:class: dropdown

We used the pattern `function[paramA_,paramB_][argument_] = ...`

Will use this pattern frequently, as it achieves two convenient tasks:
1. Separates arguments and parameters, so we can use forms like: 
      `function[paramA,paramB]/@listOfArguments`
2. Provides a convenient way of assigning symbols to expressions, as opposed to something like:
      `expressionRHS /.{paramA->.., paramB->..}`

:::

The solution is a simple sinusoidal term with a natural frequency $\omega = \sqrt{g/L}$. 

Let's animate one such pendulum, for $g = 1, L = 1, \theta_0 = \pi/12$

:::{wolfram}
:echo: true

visualizePendulum[length_, color_ : Black][angle_] := Graphics[{
   {color, Line[{{0, 0}, length {Sin[angle], -Cos[angle]}}]},
   {color, PointSize[Large], Point[length {Sin[angle], -Cos[angle]}]}
   }, PlotRange -> 1.025 {{-1, 1}, {-1, 0}}]
:::

:::{wolfram}
:echo: true
:caption: Single pendulum widget.

(*#|label:widget_single_pendulum*)
With[{g = 1, \[Theta]0 = \[Pi]/12, L = 1},
 Animate[
  visualizePendulum[L][
   smallDisplacementSymbolicSolution[\[Theta]0, g, L][t]], {t, 0, 60},
   AnimationRate -> 5, AnimationRunning -> False, Paneled -> False]
 ]
:::

It is now straight-forward to add multiple pendula with different lengths in @widget_single_pendulum!

:::{wolfram}
:echo: true
:caption: Multiple pendula widget.

(*#|label:widget_multiple_pendula*)
With[{g = 1, \[Theta]0 = \[Pi]/12},
 Animate[Show[
   Table[
    visualizePendulum[length][
     smallDisplacementSymbolicSolution[\[Theta]0, g, length][
      t]], {length, Subdivide[1/2, 1, 10]}]
   ], {t, 0, 60}, AnimationRate -> 5, AnimationRunning -> False, 
  Paneled -> False]
 ]
:::
